org-mode学习

2019.10.11

大纲

标题   title

  • 文章标题可以通过 

    #+TITLE: title-name

  • 使用*+space表示标题, 最多支持10级标题

快捷键   keybinds

折叠   fold

Key Action
S-TAB 循环切换整个文档的大纲状态 FOLDED->CHILDREN->SUBTREE
TAB 循环切换光标所在大纲的状态

移动   move

Key Action
C-c C-n/p 下/上一标题
C-c C-f/b 下/上一同级标题
C-c C-u 跳到上一级标题
C-c C-j 切换到大纲浏览状态

编辑   edit

Key Action
C-RET 插入一个同级标题
C-S-RET 插入一个同级TODO标题
M-LEFT/RIGHT 将当前标题升/降级
M-S-LEFT/RIGHT 将子树升/降级
M-S-UP/DOWN 将子树上/下移
C-c * 将本行设为标题正文
C-c C-w 将子树或区域移动到另一标题处（跨缓冲区）
C-x n s/w 只显示当前子树/返回
C-c C-x b 在新缓冲区显示当前分支（类似C-x n s)
C-c C-c 取消高亮(搜索高亮)
C-c / 创建稀疏树

链接

  1. 符合链接规则的内容，org-mode会自动将其视为链接

  2. 指定链接 

    [[link][description]]
[[link]]

  3. 文件链接 

       file:~/code/main.c::255                     进入到 255 行

  4. 跳转链接 C-c C-o

  5. 显示图片
    使用 iimage 插件来显示本地图片。 

    ;; iimage mode
(autoload 'iimage-mode "iimage" "Support Inline image minor mode." t)
(autoload 'turn-on-iimage-mode "iimage" "Turn on Inline image minor mode." t)

    使用 M-x iimage-mode RET 使用改插件
    或添加hook 

    (add-hook 'org-mode-hook 'iimage-mode)

标记语言

字体 

*粗体* /item/ +删除线+ _下划线_ 下标: H_{2}O 上标: E=mc^{2}
等款字: =git= ~git~

粗体 item 删除线 下划线 下标: H2O 上标: E=mc2
等款字: git git
特殊字符: 可以在org文件中直接使用latex语法输入特殊字符，比如希腊字符 \alpha，\beta 将显示为 α, β。不需要专门的数学环境

段落

单个回车换行的文本，默认没有换行, 需要输入两个回车, 或者输入\\表示换行

表格

使用 C-c | 或手动使用 | 生成表格

Key Action
TAB 在表格间移动
S-TAB 反向移动
C-c RET 生成表格结构，添加水平分割线并跳到下一行
RET 移动到下一行，新建一行
C-c C-c 自动调整表格
C-c - 添加水平分割线
C-c ^ 根据当前列排序，可以选择排序方式
M-LEFT/RIGHT 移动列
M-UP/DOWN 移动行
M-S-LEFT/RIGHT 删除/插入列
M-S-UP/DOWN 删除/插入行

数学公式 

\begin{equation}
x=\sqrt{b}
\end{equation}

\begin{eqnarray}
x & = & y + z \\
& = & z + t
\end{eqnarray}

If $a^2=b$ and \(b=2 \), then the solution must be 
either $$ a=+\sqrt{2} $$ or \[ a=-\sqrt{2} \].

\begin{equation} x=\sqrt{b} \end{equation}

\begin{eqnarray} x & = & y + z \\ & = & z + t \end{eqnarray}

If $a=b$ d \(b=2 \), then the solution must be either $$ a=+\sqrt{2} $$ or \[ a=-\sqrt{2} \].

宏 

#+MACRO: name replacement text $1, $2 are arguments

调用这个宏的方法是 {{{name(arg1,arg2)}}}

列表

  1. 有序列表、无序列表和描述列表。

    • 无序列表以'-'、'+'或者'*'开头

    • 有序列表以'1.'或者'1)'开头

  2. 注意事项

    • 列表符号后面都要有空格

    • 同级别的列表缩进要相同

    • 如果想要加入同级别的列表，可以 M-RET

    • 空两行之后列表结束

Key Action
TAB 折叠列表项
M-RET 插入项
M-S-RET 插入带复选框的项
M-S-UP/DOWN 移动列表项
M-LEFT/RIGHT 升/降级列表项，不包括子项
M-S-LEFT/RIGTH 升/降级列表项，包括子项
C-c C-c 改变复选框状态
C-c - 更换列表标记（循环切换）

分割线 

----- 五条短线以上位分割线

嵌入metadata 

s    #+begin_src ... #+end_src 
e    #+begin_example ... #+end_example  单行的例子以 : 开头
C    #+begin_comment ... #+end_comment  注释 单行注释以 # 开头
q    #+begin_quote ... #+end_quote      通常用于引用，与默认格式相比左右都会留出缩进
v    #+begin_verse ... #+end_verse      默认内容不换行，需要留出空行才能换行
c    #+begin_center ... #+end_center 
l    #+begin_latex ... #+end_latex 
L    #+latex: 
h    #+begin_html ... #+end_html 
H    #+html: 
a    #+begin_ascii ... #+end_ascii 
A    #+ascii: 
i    #+index: line 
I    #+include: line

上面的单字母为快捷键字母，如输入一个 <s 然后 TAB 就会扩展

注脚

在 org mode 中，你可以为你的文章添加注脚（footnote）。\\

  • 使用方括号+fn+名字 [fn:1]
    插入脚注：C-c C-x f
    C-c C-c 跳转注脚

预定义tag 

在标题后面使用 : 对标题定义tag
Org-mode的标签自动按照大纲树的结构继承。即子标题自动继承父标题的标签。

#+FILETAGS: :Peter:Boss:Secret:
使用以上定义会是文档中的所有标题都具有某些标签
Key Action
C-c \ 标签查找

导出

C-c C-e 导出文件

  • 使用指定的css导出文件 

#+HTML_HEAD: <link rel="stylesheet" type="text/css" href="/home/red/styles/markdown/markdown.css"/>
#+HTML_HEAD: <link rel="stylesheet" type="text/css" href="https://gongzhitaao.org/orgcss/org.css"/>

  • 包含文件
    导出文档时，可以包含其他文件中的内容。 

     #+INCLUDE: "~/.emacs" example
 #+INCLUDE: "~/first.py" src python
