Linux没有使用lvm合并两个相邻的分区

2019.11.05

过程

  1. 使用 df -hl 确定要合并的分区

  2. 卸载要合并的两个分区 umount /dev/sdaX

  3. 使用 fdisk /dev/sda 来删除相邻的两块分区(确定后一个分区没有数据)(删除分区数据是不会丢失的)

  4. 格式化一下没有数据的那个分区(后一块分区)

  5. 然后再将两块分区新建成一个分区, 但是现在在挂载上看分区大小没有改变, 需要重新调整分区的大小

  6. 使用 resize2fs -p /dev/sdaX 来调整分区的大小

    • 可能会出现错误提示使用 e2fsck -f /dev/sdaX 来进行磁盘检查

  7. 然后再挂载上扩容的分区就可以看到空间扩充完成
